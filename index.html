Seattle Open Device Lab

Perfect your mobile experiences

Open Device Labs are a shared community resource for testing apps and websites on a variety of phones, tablets and other devices. Instead of only testing on the few devices we personally own, or shelling out for a slightly larger set of test devices, we can build a comprehensive set of devices to share.

As far as I can tell, Seattle doesn't have an ODL yet. I'm working on fixing this. We will have a lab available to use by December 2013. Please sign up below if you'd like to be notified when all the details are in place. Your address will only be used to announce big changes to the Seattle ODL (such as when and where it initially becomes available).
